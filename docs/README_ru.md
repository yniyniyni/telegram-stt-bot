# Telegram Speech-to-Text Бот

[English version (Английская версия)](../README.md)

Бот для автоматического распознавания речи (Speech-to-Text, STT) в Telegram, созданный на **Node.js**, **TypeScript**, **SQLite** и **Deepgram API** (модель Nova-2).

Бот автоматически переводит в текст голосовые сообщения и видеосообщения (круглые видео/кружки) в авторизованных чатах, кэширует результаты для экономии бюджета API и полностью игнорирует другие сообщения для обеспечения конфиденциальности.

## Возможности

- **Автоматический STT**: Расшифровывает голосовые сообщения (`.ogg`) и видеокружки (`.mp4`) автоматически при их отправке в чат.
- **Прямое распознавание видеокружков**: Отправляет видеофайлы напрямую в Deepgram API без необходимости устанавливать и использовать `ffmpeg` на хост-системе.
- **Конфиденциальность**: В базе данных сохраняются только метаданные голосовых сообщений/кружков и текст расшифровки. Сами аудиофайлы удаляются с хоста сразу после отправки результата.
- **Fail-Closed авторизация**: Белый список чатов (`ALLOWED_CHATS` в `.env`) предотвращает использование бота посторонними лицами и защищает ваш баланс API.
- **Лимиты частоты запросов**: Встроенный ограничитель (Rate Limiter) с плавающим окном защищает бота от спама и DOS-атак.
- **Smart Formatting**: Интеграция параметров Deepgram `smart_format`, `punctuate` и `numerals` автоматически расставляет знаки препинания, абзацы, форматирует даты, числа и валюты для удобства чтения.
- **Локализация**: Интерфейс бота и заголовки поддерживают русский (`ru`) и английский (`en`) языки на основе переменной `BOT_LANGUAGE`.
- **Разделение длинных сообщений**: Санирует HTML-теги и делит длинные расшифровки на сообщения объемом до 4000 символов, предотвращая ошибки API Telegram.
- **Интеллектуальное улучшение текста (Gemini)**: Интеграция с Gemini AI (по умолчанию используется `gemini-3.1-flash-lite`) для полировки длинных расшифровок (по умолчанию >45 сек). Нейросеть автоматически исправляет ошибки, опечатки, пунктуацию, удаляет слова-паразиты и исправляет неверно распознанные слова на основе контекста, сохраняя при этом исходный порядок слов и смысл.
- **Гибкое управление полишингом**: Возможность полностью отключить улучшение текста или отключить его выборочно для видеосообщений (круглых видео / кружков) с помощью параметров в `.env`.

## Технологический стек

- **Язык**: TypeScript
- **Среда выполнения**: Node.js (v20.17.0+)
- **Фреймворк**: Telegraf (обертка Telegram Bot API)
- **База данных**: SQLite (библиотеки `sqlite` и `sqlite3`)
- **STT движок**: Deepgram REST API (модель Nova-2)
- **Улучшение текста (AI)**: Google Gemini API (библиотека `@google/genai`)

## Быстрый старт

### Требования

- Установленный Node.js (v20.17.0 или выше)
- Токен Telegram бота (можно получить у [@BotFather](https://t.me/BotFather))
- API Ключ Deepgram (можно получить в [console.deepgram.com](https://console.deepgram.com/))
- API Ключ Google Gemini (можно получить в [Google AI Studio](https://aistudio.google.com/)) (Опционально: только если включена полировка сообщений)

### Установка

1. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   cd telegram-stt-bot
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл конфигурации `.env`:
   ```bash
   cp .env.example .env
   ```
   Впишите свои ключи, укажите разрешенные ID чатов и настройте лимиты.

## Конфигурация (`.env`)

Подробное описание параметров можно найти в [.env.example](../.env.example):

- `TELEGRAM_BOT_TOKEN`: Токен вашего бота.
- `DEEPGRAM_API_KEY`: API ключ Deepgram.
- `DEEPGRAM_MODEL`: Используемая модель Deepgram (по умолчанию: `nova-2`).
- `DEEPGRAM_SMART_FORMAT`: Активация автоформатирования (по умолчанию: `true`).
- `ALLOW_ALL_CHATS`: Если `true`, бот работает во всех чатах. Если `false`, доступ разрешен только чатам из `ALLOWED_CHATS`.
- `ALLOWED_CHATS`: Разделенный запятыми список ID разрешенных чатов (например, `-100123456789,987654321`).
- `ALLOW_ALL_USERS`: Если `true`, любой пользователь может писать боту в личные сообщения (ЛС). Если `false`, доступ к ЛС разрешен только пользователям из списка `ALLOWED_USERS` (по умолчанию: `true`).
- `ALLOWED_USERS`: Разделенный запятыми список ID пользователей Telegram, которым разрешено писать боту в ЛС.
- `RATE_LIMIT_MAX_REQUESTS`: Максимальное количество расшифровок в рамках временного окна.
- `RATE_LIMIT_WINDOW_SEC`: Длительность временного окна лимита частоты запросов в секундах.
- `BOT_LANGUAGE`: Язык интерфейса (`ru` или `en`).
- `MAX_AUDIO_DURATION_SEC`: Максимальная длительность аудио для распознавания в секундах.
- `DB_FILE`: ПУТЬ к файлу базы данных SQLite.
- `DEBUG`: Включение подробных логов отладки.
- `GEMINI_POLISH_ENABLED`: Установите в `false`, чтобы полностью отключить полишинг от Gemini (по умолчанию: `true`).
- `GEMINI_POLISH_VIDEO`: Установите в `false`, чтобы отключить полишинг от Gemini специально для видеосообщений (по умолчанию: `true`).
- `GEMINI_API_KEY`: Ваш API ключ Google Gemini.
- `GEMINI_MODEL`: Модель Gemini для полишинга (по умолчанию: `gemini-3.1-flash-lite`).
- `POLISH_MIN_DURATION_SEC`: Минимальная длительность аудио/видео сообщения в секундах для запуска полишинга (по умолчанию: `45`).
